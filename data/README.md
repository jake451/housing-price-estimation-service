## Dataset

This project uses the "Seattle House Price Prediction" dataset published on Kaggle.

Due to licensing and size considerations, the full dataset is not committed to this repository.

### How to obtain the full dataset
1. Download the dataset from:
   https://www.kaggle.com/datasets/samuelcortinhas/house-price-prediction-seattle
2. Place the CSV file in:
   data/raw/

### Data layout
- data/raw/      # full dataset (ignored by git)
- data/sample/   # small sample dataset (committed)
